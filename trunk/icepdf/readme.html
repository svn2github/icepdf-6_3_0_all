Source Code and Binary Access

The ICEpdf source code is available in its entirety from the public SVN repo http://anonsvn.icesoft.org/repo/icepd. Users wishing to contribute to the project can contact product.support@icesoft.com and should fill out a contributors agreement.

ICEpdf distribution bundles can be downloaded from http://www.icesoft.org/java/downloads/icepdf-downloads.jsf.

ICEpdf distribution are uploaded the Maven central repository on a regular basis, summary information can be found here.

Building the ICEpdf library from source

The ICEpdf distribution bundle contains libs-sources folder that contains the source for the core and viewer modules. This was intended for incorporation with a developers IDE and was not designed for building the library from source.

Building the ICEpdf library from source first requires that the libraries source is checked out using a SVN client from http://anonsvn.icesoft.org/repo/icepd. Once downloaded the project can be build using Gradle or Maven.
